# big-up.sh

A comprehensive system management and hardening tool written in D.

## Features
- Health Checks
- Multi-stage System Updates
- Security Hardening
- Remote Management via PSSH

## Usage
1. Compile the project: `dub build --build=release`
2. Install: `sudo mv ./sys-manager /usr/local/bin/sys-manager`
3. Run: `sys-manager check`
